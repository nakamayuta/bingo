# 使い方
- ホスト側でビンゴの操作を行います。
- ゲスト側ではビンゴの操作はできません。
- それぞれのページのコンソールに役名とホストのURL, ゲストのURLが書かれています。
- ページの右上にあるリストアイコンでビンゴの全履歴が確認できます。

- 人数:1人 ※2人以上だと不具合が起こります。現在修正中

- 人数:複数人
